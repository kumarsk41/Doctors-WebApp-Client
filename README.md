# Doctor Appointment Web App (Frontend - Client)

## Project Overview
This is the frontend application of a Doctor Appointment System built using Thymeleaf.
It communicates with a Spring Boot REST backend using RestTemplate.

## Tech Stack

**Frontend:**
- Thymeleaf
- HTML, CSS, JavaScript

**Backend:**
- Spring Boot (REST APIs)
- Health AI Integration
- Oauth2
- Spring Security

**Database:**
- MySQL

**Tools:**
- Postman
- Maven

## Features

- View list of doctors
- Book appointments
- REST API integration
- Clean UI using Thymeleaf templates

## How to Run

### Backend
1. Start backend Spring Boot application
2. Ensure it runs on port 7071

### Frontend
1. Clone this repo
2. Open in STS
3. Run as Spring Boot app
4. Visit: http://localhost:7070
